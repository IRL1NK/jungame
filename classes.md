## Документация классов## Puzzle
Класс головоломки. Отвечает за хранение вопроса, правильного ответа и проверку пользовательского ввода для открытия заблокированных проходов.
## Поля

| Поле | Тип | Описание |
|---|---|---|
| question | std::string | Текст вопроса головоломки |
| correctAnswer | std::string | Строка с эталонным правильным ответом |

## Методы

| Метод | Назначение |
|---|---|
| solve() | Отображение вопроса, чтение ответа игрока и валидация результата |

## Player
Класс игрока. Хранит данные о текущем местоположении персонажа и управляет изменением его координат.
## Поля

| Поле | Тип | Описание |
|---|---|---|
| currentLocationId | int | Идентификатор текущей локации, где находится игрок |
| x | int | Координата персонажа по оси X |
| y | int | Координата персонажа по оси Y |

## Методы

| Метод | Назначение |
|---|---|
| getCurrentLocationId() | Получение идентификатора текущей локации игрока |
| getX() | Получение текущей координаты по оси X |
| getY() | Получение текущей координаты по оси Y |
| setPosition() | Изменение координат игрока в пределах текущей комнаты |
| changeLocation() | Перемещение игрока в другую локацию со сменой координат |

## Location
Игровая локация. Содержит структуру карты комнаты, управляет её отрисовкой и логикой доступа.
## Поля

| Поле | Тип | Описание |
|---|---|---|
| id | int | Уникальный идентификатор локации |
| map | std::vector<std::string> | Двумерная ASCII-карта помещения |
| isOpen | bool | Флаг доступности локации для игрока (открыта/закрыта) |
| puzzle | Puzzle* | Указатель на объект головоломки, блокирующей проход (при наличии) |

## Методы

| Метод | Назначение |
|---|---|
| getId() | Получение уникального идентификатора локации |
| getIsOpen() | Проверка, открыта ли локация для входа |
| openLocation() | Изменение статуса локации на открытый |
| getTile() | Получение символа карты по заданным координатам |
| render() | Вывод карты локации в консоль с учетом позиции игрока |
