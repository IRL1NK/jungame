# Описание методов

## Puzzle::solve()

Отображает загадку игроку, запрашивает ввод и проверяет правильность ответа.

### Параметры

Отсутствуют.

### Возвращаемое значение

| Тип | Описание |
| :--- | :--- |
| bool | true при правильном ответе, false при неверном |

## Player::setPosition(int newX, int newY)

Устанавливает новые абсолютные координаты игрока на текущей локации.

### Параметры

| Имя | Тип | Описание |
| :--- | :--- | :--- |
| newX | int | Новая координата по оси X |
| newY | int | Новая координата по оси Y |

### Возвращаемое значение

Отсутствует.

## Player::changeLocation(int newLocId, int newX, int newY)

Перемещает игрока в другую локацию и задает его новые стартовые координаты в ней.

### Параметры

| Имя | Тип | Описание |
| :--- | :--- | :--- |
| newLocId | int | Идентификатор целевой локации |
| newX | int | Начальная координата по оси X в новой комнате |
| newY | int | Начальная координата по оси Y в новой комнате |

### Возвращаемое значение

Отсутствует.

## Location::getTile(int targetX, int targetY)

Возвращает символ карты (тайл) по указанным координатам с проверкой выхода за границы.

### Параметры

| Имя | Тип | Описание |
| :--- | :--- | :--- |
| targetX | int | Целевая координата по оси X |
| targetY | int | Целевая координата по оси Y |

### Возвращаемое значение

| Тип | Описание |
| :--- | :--- |
| char | Символ элемента карты (пробел, стена или ID двери) |

## Location::render(int playerX, int playerY)

Отрисовывает карту локации в консоль с учетом текущего положения игрока.

### Параметры

| Имя | Тип | Описание |
| :--- | :--- | :--- |
| playerX | int | Текущая координата игрока по оси X |
| playerY | int | Текущая координата игрока по оси Y |

### Возвращаемое значение

Отсутствует.
